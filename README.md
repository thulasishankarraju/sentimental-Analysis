# sentimental-Analysis
# Sentiment Analysis Web Application

## Overview
This project is a simple AI-based Sentiment Analysis Web Application developed using Flask and TextBlob.  
The application analyzes user text and predicts whether the sentiment is:

- Positive 😊
- Negative 😞
- Neutral 😐

The project also stores user reviews in a CSV file and generates graphical sentiment reports using Matplotlib.

## Technologies Used

- Python
- Flask
- TextBlob
- Pandas
- Matplotlib
- HTML/CSS

## Features

- User-friendly web interface
- Real-time sentiment prediction
- Automatic polarity detection
- CSV data storage
- Sentiment graph generation
- Positive, Negative, Neutral classification
